# Bulk Phone Lookup & Verification App (চেকদ্যাটফোন বাল্ক ডেমো)

এটি একটি ফুল-স্ট্যাক Node.js এবং Express অ্যাপ্লিকেশন যা `checkthatphone.com`-এর `/v1/lookup-demo` এপিআই ব্যবহার করে রিয়েল-টাইমে একসাথে একাধিক ফোন নম্বর চেক করার সুবিধা প্রদান করে। 

অ্যাপ্লিকেশনটি একটি আধুনিক এবং প্রিমিয়াম **লাইট থিম (Light Theme)** ইন্টারফেসে তৈরি করা হয়েছে।

---

## কিভাবে রান করবেন (Setup & Execution Guide)

Cloudflare Turnstile উইজেটটি নির্দিষ্ট ডোমেইনে সীমাবদ্ধ (Domain restricted) থাকার কারণে, স্থানীয়ভাবে (localhost-এ) এটি রান করতে হলে নিচের নির্দেশনাবলী অনুসরণ করতে হবে:

### ১. হোস্টস ফাইল কনফিগারেশন (Configure Hosts File)
উইন্ডোজের হোস্টস ফাইলে লোকালহোস্টকে `dev.checkthatphone.com` ডোমেইনে ম্যাপ করুন:
1. উইন্ডোজের সার্চ বারে **Notepad** লিখে সার্চ করুন।
2. Notepad আইকনে রাইট-ক্লিক করে **Run as administrator** সিলেক্ট করুন।
3. Notepad-এ ফাইলটি ওপেন করুন: `C:\Windows\System32\drivers\etc\hosts`
4. ফাইলের একদম নিচে নিচের লাইনটি যুক্ত করুন এবং সেভ করুন:
   ```text
   127.0.0.1 dev.checkthatphone.com
   ```

### ২. ডিপেন্ডেন্সি ইনস্টলেশন (Install Dependencies)
প্রজেক্টের মূল ডিরেক্টরিতে কমান্ড লাইনে রান করুন:
```bash
npm install
```

### ৩. সার্ভার চালু করা (Start Server)
নিচের কমান্ডের মাধ্যমে এক্সপ্রেস ব্যাকএন্ড প্রক্সি সার্ভারটি চালু করুন:
```bash
npm start
```
সার্ভারটি সফলভাবে `http://dev.checkthatphone.com:8888/` ঠিকানায় চালু হবে।

### ৪. ব্রাউজারে রান করুন (Access in Browser)
ব্রাউজারে যান এবং এই ইউআরএলটি ওপেন করুন:
👉 **[http://dev.checkthatphone.com:8888/](http://dev.checkthatphone.com:8888/)**

## অ্যাপটির মূল ফিচারসমূহ (Key Features)

- **লাইট থিম ইউজার ইন্টারফেস (Premium Light Theme UI)**: চমৎকার মিনিমালিস্ট ডিজাইন এবং কার্ড ট্রানজিশন অ্যানিমেশন।
- **একসাথে একাধিক নম্বর চেক (Multiple Phone Inputs)**: টেক্সট এরিয়াতে কমা (`,`) অথবা নিউলাইন দিয়ে আলাদা করে একাধিক ফোন নম্বর একসাথে ইনপুট দিতে পারবেন।
- **স্বয়ংক্রিয় ব্যাচ প্রসেসিং (Automated Batch Processing)**: এপিআই প্রতি রিকোয়েস্টে নতুন Turnstile টোকেন দাবি করে। আমাদের ক্লায়েন্ট কোড স্বয়ংক্রিয়ভাবে Turnstile উইজেটটি রিসেট করে প্রতিবার নতুন টোকেন সংগ্রহ করে একের পর এক নম্বর প্রসেস করে।
- **ডিটেইলস ভিউয়ার (Details Report Drawer)**: ভ্যালিডেশন শেষে যেকোনো নম্বরের পাশে **Details** বাটনে ক্লিক করলে ক্যারিয়ার নেটওয়ার্কের নাম, পোর্ট স্ট্যাটাস, জিপিএস জিওলোকেশন এবং TCPA আইনি ঝুঁকির (Litigator risk) তথ্য চমৎকার লেআউটে ভেসে উঠবে।
- **রিসোর্স ভিউয়ার (Raw JSON output)**: ডেভেলপারদের সুবিধার্থে সরাসরি এপিআই থেকে প্রাপ্ত রিসিভড ডেটা JSON ফরম্যাটে হাইলাইটেড কোডবক্সে প্রদর্শন করা হবে।


১. ক্রোম ব্রাউজারে এই লিঙ্কে প্রবেশ করুন: 👉 chrome://net-internals/#hsts
২. পেজের একদম নিচে "Delete domain security policies" সেকশনটি খুঁজে বের করুন।
৩. ইনপুট বক্সে checkthatphone.com লিখুন এবং Delete বাটনে ক্লিক করুন।
৪. একইভাবে ইনপুট বক্সে dev.checkthatphone.com লিখুন এবং Delete বাটনে ক্লিক করুন।
৫. এবার ক্রোমের ব্রাউজিং ক্যাশ/হিস্ট্রি ক্লিয়ার করুন (অথবা একটি Incognito (ছদ্মবেশী) উইন্ডো ওপেন করুন)।
৬. এবার ক্রোমে আবার এই লিঙ্কটি ট্রাই করুন: http://dev.checkthatphone.com:8888/ (ইউআরএল-এর শুরুতে অবশ্যই http:// লিখবেন, https:// নয়)।
